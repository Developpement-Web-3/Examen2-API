# Examen2

Récupérez le projet 
```console
git clone https://github.com/Developpement-Web-3/Examen2-API.git
```

Déplacez-vous dans le dossier du projet
```console
cd Examen2-API
```

  
Installez les modules 
```console
npm install
```
  
Vous êtes prêts à démarrer votre API sur MongoDB!
